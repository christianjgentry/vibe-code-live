# Vibe Code Live

A live coding entertainment event where developers build apps from audience ideas.

**[View the live site →](https://YOUR_USERNAME.github.io/vibe-code-live/)**

## What's This?

Vibe Code Live is a tech-focused bar event concept featuring:
- A developer who "vibe codes" an app live on stage
- An MC who explains concepts and engages the audience
- Audience participation through voting and Q&A
- Themed drinks, trivia, and prizes

## Files

- `index.html` - The main explainer/landing page with interactive revenue calculator
- `VibeCodLive_EventModel.xlsx` - Financial model spreadsheet

## Deploy Your Own

1. Fork this repo
2. Go to Settings → Pages
3. Set source to "Deploy from a branch" → `main` → `/ (root)`
4. Your site will be live at `https://YOUR_USERNAME.github.io/vibe-code-live/`
